阿里云虚拟机建站（mac）

微信开发页面，没有备案老是被微信 110 拦截，只好买个阿里虚拟机，并且去备个案玩玩。

购买主机，送域名

阿里云虚拟主机购买

  • 最便宜的，一年 298，送一年 .xin 域名优惠券，记得去买，有效期没几天；

阿里云域名购买

  • 直接支付，付款时选择优惠券；

虚拟机控制台配置

  • 进入管理；
  • 站点信息，设置一些密码；
  • 域名绑定，需要在域名控制台对应配置，并且还需要备案；
  • 工具&服务：可以检查网站的一些配置情况；
  • 上传网站文件：FTP

关于 mac 使用 ftp

  • Finder - 前往 - 连接服务器（⌘ + K）
  • 服务器地址：ftp：//192.xxx.xxx.xxx
  • 输入刚刚填写的那些密码，使用 ftp 登录用户名与密码，即可连接；
  • 在 htdocs 文件夹下，复制 网站文件即可；
  • Finder 只能查看，不能操作；

使用下面终端方法

  • 终端：cd 到本机网站文件夹；
  • ftp 132.xxx.xxx.xxx
  • 用户名
  • 密码
  • dir
  • cd htdocs（linux 服务器）
  • put xxx.zip
  • 阿里虚拟机管理后台解压完成

关于 windows 使用 ftp

  • 我的电脑：ftp://142.xxx.xxx.xxx
  • 用户名
  • 密码
  • ok，比 mac 方便多了

关于文件夹的删除

  • 需要在虚拟机管理后台删除

域名解析

  • 添加网站主机记录，默认 www ，ip 填写之前购买的虚拟机 ip 即可；

网站备案

  • 阿里云管理控制台 - 右上角直接去备案（不备案，没法打开网站）
  • 填写资料即可，等待审核；
