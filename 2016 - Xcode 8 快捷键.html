Xcode 快捷键

File

  • 新建 New
command + T ： 新建标签
shitf + command + T ： 新建窗口
command + N ： 新建文件
shift + command + N ： 新建项目
option + command + N ： 新建Group
  • 打开 Open
command + O ： 打开电脑中的文件
shift + command + O ： 快速搜索项目中文件并打开
option + command + A ： 添加项目文件 add Files
  • 关闭 Close
command + W ： 关闭标签
option + command + W ： 关闭项目

Edit

  • 基本
command + C/V/X/A/Z ： 赋值/粘贴/剪切/全选/撤销
shift + command + Z ： 反撤销
command + D ： 删除此行

View

  • 编辑区域 Editor
command + 回车 ： 标准编辑页面
option + command + 回车 ： 双屏编辑页面

control + 1 ： 相关文件
control + 2 之前的关系文件（与control + command + 2 相同）
control + 3 之后的关系文件

control + 4 项目文件
control + 5 当前文件所在group 的所有文件
control + 6 当前文件所有方法
  • 项目目录区域 Navigators
command + 0 ： 隐藏/显示 左侧
command + 1~8 ： 对应切换标签
  • Debug 区域
shift + command + C ： 开启控制台？
shift + command + Y ： 隐藏/显示 Debug
  • 信息区域：Utilities 
option + command + 0 ：  隐藏/显示 右侧
option + command + 1~6 ： 切换右侧上面的标签（code下2个，xib下6个）
control + option + 1~4 ： 切换右侧下面的标签
control + option + command + 1~4 ： 对应上面4个的快速输入激活
  • 其他
option + command + T ： 隐藏/显示 顶部ToolBar 
control + F ： 全屏

Find

  • 当前文件搜索
command + F 查找当前页的内容，使用回车可以循环下一个
option + command 替换当前页内容 回车可以循环下一个
command + G 下一个
shift + command + G 上一个
  • 项目搜索
shitf + command + F 全局搜索 
shift + control + command + F 对选中的内容直接搜索
shift + option + command + F 全局替换
control + command + G 查找下一个 
shitf + control + command + G 查找上一个

Navigate

  • 都很好用
shift + command + J ： 显示当前文件所在项目位置
command + J ： 选择当前活跃区域
control + command + J ： 前往该类（与command + 鼠标左键一致）

control + 左/右 ： 向前向后切换文件（与鼠标单指左右滑动效果一致）

control + 上/下 ： 当前.h.m 切换
shift + control + command + 上/下 双屏展示该文件的.h.m

command + L ： 跳转指定行
shift + command + L ： 滚动到当前编辑行

Editor

  • 编辑
control + command + E ： 选中内容，当前页加黑显示，一处修改，全部修改！
command + 【/】： 左右缩进
command + / ： 整行注释
option + command + / ： 文档注释

option + command + 左/右 ： 折行/展开
shift + option + command + 左/右 ： 当前页所有函数方法折行/展开

Product

  • 一般
command + B ： 编译
command + R ： 运行
Command + . ： 停止
command + U ： Test
command + I ： Profile

shirt + command + B ： Analyze
Command + ： ： 检查拼写
shift + command + K ： 清理

shift + command + < ： 打开shame
  • 条件编译 运行
shift + command + A 安装？

shift + command + R 只编译为了运行
shift + command + U 只编译为了Test
shift + command + I 只编译为了Profile

control + command + R 不编译直接运行，对应上面的使用
control + command + U 不编译直接Test，对应上面的使用
control + command + I 不编译直接Profile，对应上面的使用
 

Debug

command + Y 打开关闭断点功能（所有）
command + \ 改行加/减断点
shift + command + \ 加符号断点

F6 ： 单步
F7 ： 跳入
F8 ： 跳出
control + command + Y ： 继续
control + command + C ： 继续直到，当前选择行

Window

command + 0 ： 帮助文档
command + 1 ： 再次打开（进入）xcode

使用 storyboard / xib 在 Editor 中会有更多帮助

command + +/- ： 放大/缩小
control + command + = 原始大小
// 更多...

其他

  option + commamd + D 显示隐藏Dock
