关于 Swift

  • Swift是一种新的编程语言，基于C和OC，采用安全类型的编程模式，支持原有的cocoa框架，目标重新定义软件开发。
  • 苹果基于原有的开发经验，比如OC中的ARC，block，framework等，都将推进Swift在未来等发展。
  • Swift类似OC等命名以及动态对象，无缝链接现有cocoa框架，支持混编OC，C，C++，
  • Swift的未来，被十分重视与看好。
  • 总之，大概就是说，Swift是十分优秀的编程语言，将在未来占有一席之地，赶紧好好学习！

枚举与结构体

  • Enumerations 基础枚举大致差不多，特殊在于可以添加函数
enum Rank:UInt {
    case ace = 1
    case two, three, four, five, six, seven, eight, nine, ten
    case jack, queen, king
    
    
    func desc() -> String {
        switch self {
        case .ace:
            return "Ace"
            
        case .jack:
            return "Jack"
            
        case .queen:
            return "Queen"
            
        case .king:
            return "King"
            
        default:
            return String(self.rawValue)
        }
    }
}

let ace = Rank.ace
let aceIndex = Rank.ace.rawValue
let aceDecs = Rank.ace.desc()

// 使用指定值获取枚举值时，是可选类型
if let card = Rank(rawValue: 3) {
  print(card.desc())
}

协议与拓展

  • Protocols
protocol TestProtocol {
    var desc:String {get}
    mutating func adjust()
}

class testClass: TestProtocol {
    var desc: String = "test class"
    func adjust() {
        desc = desc + "1"
    }
}

struct 和 enum 都可以使用协议

let test = testClass()
test.adjust()
let desc = test.desc
  • Extensions
extension Int: TestProtocol {
    var desc: String {
        return "The number \(self)"
    }
    mutating func adjust() {
        self += 1
    }
}

// 定义异常类型
enum myError: Error {
  case error1
}
   
// 可能抛出异常的函数，添加异常抛出
func testErrorFunc() throws {
  throw myError.error1
}
   
   
// do - try - catch：需要补充一个空catch，这里不太理解，待 Error Handling 部分详细研究
do {
  try testErrorFunc()
  print("ok")
} catch myError.error1 {
  print("error")
} catch {
  print("?????")
}
  • 函数相关，感觉跟异常处理无关啊 使用defer 延迟处理
var fridgeIsOpen = false
let fridgeContent = ["milk", "eggs", "leftovers"]
 
func fridgeContains(_ food: String) -> Bool {
    fridgeIsOpen = true
    defer {
        fridgeIsOpen = false
    }
    
    let result = fridgeContent.contains(food)
    return result
}
fridgeContains("banana")
print(fridgeIsOpen)

泛型

泛型，指定类型。

// 指定了数组元素是 String
var names = Array<String>()


// 定义函数申明了一种类型T，一般也用V，U等，参数与返回指定了类型T。这里就是说我入参什么类型，返回就是什么类型
func makeArray<T>(info: T, num: Int) -> [T] {
  
  var result = [T]()
  for _ in 0..<num {
      result.append(info)
  }
  
  return result
}
